Download recomended versions of QGIS (3.10 or 3.14)

Models were tested on both Windows and Linux opertaing systems.

1- Download the Models folder to your pc.

2- Open QGIS > Go to processing toolbox > click on Model and select "Add scripts to toolbox"

3- add the scrips one by on..the files names and locations will be set automatically.
